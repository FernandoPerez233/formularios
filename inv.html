🎯 CSS Grid con Propiedades de Posicionamiento

📋 Introducción

Este documento presenta una investigación completa sobre CSS Grid y cómo se integra con las propiedades de posicionamiento que también se utilizan en Flexbox. CSS Grid comparte muchas propiedades con Flexbox para el control del posicionamiento y alineación de elementos.

1. Grid Básico

Empezamos con un grid básico usando display: grid. Las propiedades fundamentales incluyen:

  • grid-template-columns: Define las columnas
  • grid-template-rows: Define las filas
  • gap: Espacio entre elementos (también usado en flex)
.grid-basic { display: grid; grid-template-columns: repeat(3, 1fr); grid-template-rows: repeat(2, 100px); gap: 10px; }

2. Alineación del Contenedor (justify-content y align-content)

Estas propiedades también se usan en Flexbox y controlan la alineación de toda la grilla dentro de su contenedor:

.grid-alignment { display: grid; grid-template-columns: repeat(3, 80px); grid-template-rows: repeat(2, 60px); gap: 10px; height: 200px; justify-content: center; /* Centra horizontalmente */ align-content: space-around; /* Distribuye verticalmente */ }

3. Alineación de Items (justify-items y align-items)

align-items también se usa en Flexbox. Estas propiedades controlan cómo se alinean los elementos dentro de sus celdas de grid:

.grid-items-alignment { display: grid; grid-template-columns: repeat(3, 120px); grid-template-rows: repeat(2, 80px); justify-items: center; /* Centra items horizontalmente en su celda */ align-items: end; /* Alinea items al final de su celda */ }

4. Auto-alineación (justify-self y align-self)

align-self también se usa en Flexbox. Permiten que elementos individuales se alineen de manera diferente:

.item-1 { justify-self: start; /* Se alinea al inicio */ align-self: start; width: 60px; height: 60px; } .item-2 { justify-self: center; /* Se centra */ align-self: center; width: 80px; height: 40px; } .item-3 { justify-self: end; /* Se alinea al final */ align-self: end; width: 70px; height: 70px; }

5. Grid con Posicionamiento Absoluto

Los elementos con position: absolute pueden coexistir con grid. El contenedor grid actúa como contexto de posicionamiento:

.grid-positioning { display: grid; position: relative; /* Contexto para position: absolute */ } .absolute-item { position: absolute; top: 10px; right: 10px; z-index: 10; }

6. Grid con Posicionamiento Sticky

position: sticky funciona perfectamente dentro de contenedores grid para crear elementos que se "pegan" durante el scroll:

.grid-sticky { display: grid; grid-template-columns: 200px 1fr; height: 300px; overflow-y: auto; } .sticky-sidebar { position: sticky; top: 0; height: fit-content; }

7. Grid + Flexbox Híbrido

Grid y Flexbox se pueden combinar perfectamente. Grid para el layout general y Flexbox para la alineación interna:

.grid-flex-hybrid { display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: auto 1fr auto; } .header { display: flex; /* Flexbox dentro de grid */ justify-content: space-between; align-items: center; } .sidebar { display: flex; flex-direction: column; /* Columna vertical */ gap: 10px; }

8. Place-items y Place-content (Propiedades Abreviadas)

Estas son propiedades abreviadas que combinan las propiedades de alineación:

  • place-items: Combina align-items y justify-items
  • place-content: Combina align-content y justify-content
.grid-place { display: grid; grid-template-columns: repeat(3, 100px); grid-template-rows: repeat(2, 80px); height: 250px; place-content: center; /* = align-content: center; justify-content: center; */ place-items: center; /* = align-items: center; justify-items: center; */ }

🎯 Valores Comunes para Propiedades de Alineación

justify-content y align-content:

  • start - Al inicio
  • end - Al final
  • center - Centrado
  • space-between - Espacio entre elementos
  • space-around - Espacio alrededor de elementos
  • space-evenly - Espacio uniforme
  • stretch - Estirar para llenar

align-items, justify-items, align-self, justify-self:

  • start - Al inicio de la celda
  • end - Al final de la celda
  • center - Centrado en la celda
  • stretch - Estirar para llenar la celda
